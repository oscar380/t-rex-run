# T-rex runner

This section explain this little javascript game

## How to play

- Press `Space` to start the game

- Press `Space` to jump

- Press `Down` to duck

-refresh the page to restart the game

## How to install

- Clone the repository

- Open the `index.html` file in your browser

## How to contribute

- Fork the repository

- Make a pull request

- Wait for the merge
